## 目标：
  在单文件中移动的更快
## 任务点：
  1. 滚动
      Ctrl + f(forward)：向下滚动一屏  
      Ctrl + b(backward)：向上滚动一屏  
      Ctrl + d: 向下滚动半屏
      Ctrl + u: 向上滚动半屏
      Ctrl + e: 向下滚动一行
      Ctrl + y: 向上滚动一行
    
  2. 基于配置（容易记忆推荐）
      shift + k: 向上滚动5行
      shift + j: 向下滚动5行
  3. 配置
     normal（normal模式）:
      ```json
          "vim.normalModeKeyBindings": [
              { // 向下移动5行
                  "before": ["J"],
                  "after": ["5", "j"]
             },
              { // 向上移动5行
                  "before": ["K"],
                  "after": ["5", "k"]
              }
          ]
      ```
      visual(可视化模式):
      ```json
        "vim.visualModeKeyBindings": [
            { // 向下移动5行
                "before": ["J"],
                "after": ["5", "j"]
            },
            { // 向上移动5行
                "before": ["K"],
                "after": ["5", "k"]
            }
        ],
      ```

  4. 常用操作
    zz: 将当前行置于屏幕中央
    zt(top): 将当前行置于屏幕顶部
    zb(bttom): 将当前行置于屏幕底部

    gg: 跳转到文件首行首字母
    G: 转到文件尾行首字母

    行数 + gg: 跳转至指定行
    行数 + G: 跳转至指定行
