退出insert方式有Esc改成jj
```json
    "vim.insertModeKeyBindings": [
        {
            "before": ["j", "j"],
            "after": ["<Esc>"]
        }
    ]
```

normal模式快速进入行首和行未
```json
    "vim.normalModeKeyBindings": [
        { // 行首
            "before": ["H"],
            "after": ["^"]
        },
        { // 行未 
            "before": ["L"],
            "after": ["g", "_"]
        },
        { // 向下移动5行
            "before": ["J"],
            "after": ["5", "j"]
        },
        { // 向上移动5行
            "before": ["K"],
            "after": ["5", "k"]
        }
    ]
```

操作符配置
```json
 "vim.operatorPendingModeKeyBindings": [
        {
            "before": ["H"],
            "after": ["^"]
        },
        {
            "before": ["L"],
            "after": ["g", "_"]
        } 
    ],
```
